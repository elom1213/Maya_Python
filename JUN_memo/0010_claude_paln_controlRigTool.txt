  - 레이어 분리: UI(main_window.py)는 cmds.*를 직접 호출하지 않고, 모든 씬 조작은 MayaScene
  어댑터를 경유. 매칭/IK 알고리즘은 RigMatcher/CageModel에 격리.
  - PySide UI: maya.cmds 의 frameLayout/textScrollList/checkBoxGrp/button →
  QGroupBox/QListWidget/QCheckBox/QPushButton. 8개 체크박스 순서는 cage 의 idx 순서와 일치하도록
  유지. 모든 UI 문자열은 영어.
  - Framework.qt.qt 추상화 + ThemeManager.load_theme_to_widget(window, "red"). dev reload 는
  "tools" 와일드카드로 자동 등록.
  - 깊은 재작성 정리: is 문자열 비교→==, State : Sucess→State : Success, 전역
  JUN_cage_glo→self.cage 캡슐화, 미사용 import maya.mel 제거. 알고리즘(orient 축 순서·pole 축·joint
  chain 인덱스·거리 attribute)은 동등하게 이식.
